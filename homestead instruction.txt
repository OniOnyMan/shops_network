1. Ставь Virtual Box
2. Ставь Vagrant 2.x

3. Клонируй репозиторий с homestead

4. Выполни команду bash init.sh
5. Редактируй созданный Homestead.yaml
	
	a. folders (from - локальная, to - виртуальная)
	
	b. sites (from - алиас, to - откуда берём)

6. Делай vagrant up
7. Если ошибок не возникло - ставь vagrant box add laravel/homestead (в условиях РКН делай это через VPN)

8. Клонируй репозиторий из ветки develop в указанный тобой путь

9. Соединяйся с вируталкой из директории Homestead посредством vagrant ssh

10. Переходи в директорию с проектом на виртуалке

11. Уставнови зависимости с помощью composer (composer install)

12. Установи npm по Женьковской ссылке

13. Экспортируй БД с оффсайта из adminer.php

14. Импортируй БД в adnimer.php своего проекта

15. Запускай миграции php artisan migrate

16. Запускай по алиасу
